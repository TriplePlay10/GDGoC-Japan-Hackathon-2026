# ドレミえる

『ドレミ』が『みえる』自動譜読みツールです。
お手持ちの楽譜画像を読み込み、マーカーでなぞるだけで音階（ドレミ）を自動判定して表示します。

## プロダクトの概要
ピアノや楽器の初心者にとって、楽譜の「譜読み」は大きな壁です。
「ドレミえる」は、ユーザーがアップロードした楽譜画像に対して、ブラウザ上で直感的に音階を可視化できるサポートツールです。

## 主な機能
* **画像読み込み**: JPG / PNG / WebP 形式の楽譜画像をドラッグ＆ドロップで読み込み可能
* **自動解析（マーカー機能）**: 五線の周辺をマーカーでなぞるだけで、黒い音符を自動で検出し「ドレミ」を振ります
* **手動編集**: 消しゴムでの修正や、クリックによる音符の手動追加・削除に柔軟に対応
* **クラウドセーブ**: Firebaseを用いた匿名ログインにより、作業の進捗を自動でクラウドに保存でき、ページを閉じても途中から再開できます
* **画像エクスポート**: ドレミが振られた状態の画像をPNGとしてダウンロード可能

## 使用技術 (Tech Stack)
* **Frontend**: Next.js (React), TypeScript, Tailwind CSS
* **Backend/BaaS**: Firebase (Authentication, Cloud Firestore, Cloud Storage)
* **画像解析**: HTML5 Canvas API によるフロントエンドでの独自ピクセル走査アルゴリズム

## ローカル開発環境のセットアップ

本プロジェクトは Next.js と Firebase を用いたサーバーレス構成を採用しています。

### 1. 前提ツール
* [Node.js](https://nodejs.org/) (v18 以上推奨)
* npm (Node.jsに付属)

### 2. インストール
リポジトリをクローン後、以下のコマンドを実行して開発サーバーを立ち上げてください。

```bash
npm install
npm run dev
```
ブラウザで http://localhost:3000 にアクセスするとアプリが起動します。
